BEXML provides fast, lazy, RESTful fastcgi and library access to various issue
(bug) trackers. It is optionally compilable into a fast binary with IronPython
and PyPy for even faster access. It lets you work with the supported issue
trackers using a single, unified interface.

Supported issue tracker backends:

* Bugs Everywhere distributed bug tracker on-disc format (file://)
* Bugs Everywhere distributed bug tracker XML format (file://, http://)
* Redmine (coming shortly)
* Github (coming shortly)
